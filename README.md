# 📄 Генератор PDF (Offline PDF Generator)

**Бесплатный офлайн-инструмент для создания PDF-документов с полной поддержкой кириллицы и режима Large Print для слабовидящих.**

![PDF Generator](screenshot.png)

## 🌟 Возможности

✅ **Полная поддержка кириллицы** — русский, украинский, белорусский и другие языки  
✅ **Large Print** — размер шрифта от 10 до 36 pt для слабовидящих  
✅ **Работает офлайн** — ни один байт данных не покидает ваше устройство  
✅ **Нумерация страниц** — автоматическая, "Страница X из Y"  
✅ **Загрузка файлов** — поддержка `.txt`, `.md` через кнопку или drag & drop  
✅ **Сохранение текста** — экспорт вашего текста в `.txt` с BOM для Windows  
✅ **Красная строка** — автоматический отступ первой строки абзаца  
✅ **Умный перенос слов** — текст не обрезается и не теряется  
✅ **Горячие клавиши** — `Ctrl+Enter` (PDF), `Ctrl+S` (текст), `Ctrl+O` (файл)  
✅ **Красивый UI** — 3D-кнопки с подсветкой, анимированный индикатор загрузки  

## 🚀 Быстрый старт

1. Скачайте файл `index.html` и библиотеку `jspdf.umd.min.js`
2. Откройте `index.html` в любом современном браузере
3. Введите текст или загрузите `.txt` файл
4. Нажмите **"📄 Создать PDF"** — готово!

**Никакой установки. Никакого интернета. Никакой регистрации.**




## 🎯 Для кого этот проект?

- 📖 **Слабовидящие пользователи** — режим Large Print (22-36 pt)
- 👵 **Пожилые люди** — крупный читаемый шрифт
- 📚 **Студенты и преподаватели** — быстрое оформление документов
- 💼 **Офисные работники** — конвертация текста в PDF без установки Word
- 🔒 **Забота о приватности** — данные не уходят на серверы

## 🛠 Технологии

- **HTML5 + CSS3 + Vanilla JavaScript** — без фреймворков
- **jsPDF** — генерация PDF-файлов
- **Canvas API** — рендеринг текста с системными шрифтами ОС
- **FileReader API** — загрузка файлов с компьютера
- **Drag & Drop API** — перетаскивание файлов

## 🔐 Приватность

Этот инструмент **полностью автономный**:
- ❌ Не отправляет данные на серверы
- ❌ Не использует внешние API
- ❌ Не собирает статистику
- ❌ Не требует интернета после загрузки
- ✅ Работает прямо в вашем браузере
- ✅ Открытый исходный код — проверяйте сами!

## ⌨️ Горячие клавиши

| `Ctrl + Enter` | Создать PDF |

| `Ctrl + O` | 📂 Загрузить файл |
